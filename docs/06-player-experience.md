# 玩家体验曲线

## 6.1 前30分钟体验节奏

```
时间轴  │ 内容                                           │ 情绪
───────┼────────────────────────────────────────────────┼──────────
0-3min │ [1-1] 教学：单槽位二选一，理解槽位概念          │ 好奇
3-6min │ [1-2] 双槽位：需要同时切换两个槽位               │ 思考
6-10min│ [1-3] 三槽位：理解出口位置与配置的关系          │ 挑战
10-15min│ [1-4] 首次失败教学：按R重置，回顾观察          │ 顿悟
15-20min│ [1-5] 第一章结尾：引入CycleSlot                 │ 成就感
20-30min│ [2-1~2-2] 第二章开始：ConditionalSlot          │ 新鲜感
```

## 6.2 "顿悟时刻"设计

顿悟时刻指玩家突然理解某个机制的关键瞬间：

| 章节 | 顿悟时刻 | 设计方式 |
|------|----------|----------|
| Ch1 | "原来只是换个方向" | 第一次切换后发现墙变路，路径就在身后 |
| Ch2 | "原来要先开这个" | 门被另一槽位挡住，需要先切换那一个 |
| Ch3 | "表面看是这样" | 伪装地板/真假墙，需要透过表象看本质 |

## 6.3 机制引入节奏

```
第一章 │ 引入:ToggleSlot(双选)→双槽位联动→三槽位组合
───────┼────────────────────────────────────────────
第二章 │ 引入:ConditionalSlot(条件解锁)→时序依赖→门控
───────┼─────────────────────────────────────────────
第三章 │ 引入:联动开关→伪装地板→空间欺骗→综合挑战
```

## 6.4 无战斗类银河恶魔城的设计哲学

**为什么无战斗？**
- 降低开发复杂度（无AI、无战斗平衡、无数值设计）
- 将认知压力集中在空间逻辑上，而非操作压力
- 适合解谜玩家核心诉求：安静思考

**与传统类银河恶魔城的区别：**

| 维度 | 传统类银河恶魔城 | 本游戏 |
|------|----------------|--------|
| 驱动类型 | 能力成长驱动（获得新能力→探索新区域） | **空间逻辑驱动**（切换房间拓扑→打通道路） |
| 进度机制 | 能力解锁门控 | 房间完成度解锁 |
| 玩家成长 | 数值/技能成长 | 无成长，纯智力挑战 |
| 探索感 | 地图逐渐展开的惊喜感 | 房间内部的拓扑重构惊喜感 |

**在没有"成长驱动"的情况下保持吸引力：**
1. **机制层层叠加：** 每章引入新机制，后一个房间需要同时运用已有+新机制
2. **拓扑美学：** 房间切换带来的空间重构感本身就是奖励（类似Gorogoa的"画卷展开"体验）
3. **无压力探索：** 无死亡惩罚，允许无限重置，降低进入门槛
4. **叙事碎片：** 通过环境物品（文件、照片、涂鸦）暗示故事背景，满足叙事需求
