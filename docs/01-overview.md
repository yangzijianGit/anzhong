# 《暗室》游戏概述

## 一句话介绍

**《暗室》**是一款2D类银河恶魔城无战斗房间解谜游戏。玩家通过切换房间内的"预制件槽位"（SwitchSlot）来重塑房间结构，开辟通往出口的道路。无敌人、无战斗、无升级成长，核心体验是"空间逻辑推理 + 房间拓扑变换"。

## 基本信息

| 要素 | 内容 |
|------|------|
| **游戏名称** | 《暗室》（暂定） |
| **类型** | 2D类银河恶魔城 · 无战斗房间解谜 |
| **目标用户** | 解谜游戏爱好者、独立游戏玩家、类银河恶魔城轻度玩家 |
| **原型参考** | Hollow Halls（无战斗探索解谜）、Dead Cells（房间预制拼贴）、Gorogoa（空间切换解谜） |

## 核心特色

玩家通过切换房间内的"预制件槽位"（SwitchSlot）来重塑房间结构，从而开辟通往出口的道路。

- **无战斗** — 降低开发复杂度，将认知压力集中在空间逻辑上
- **无成长** — 纯智力挑战，所有机制在第一章逐步引入
- **无死亡惩罚** — 可无限切换，可随时重置（Reset）房间
- **即时反馈** — 每次切换后，玩家立刻能看出道路是否打通

## 关于原型参考的说明

> **关于 Hollow Halls 的调研结果：** 它是一款**恐怖氛围**的短篇独立游戏，聚焦**故事与氛围**，非战斗导向。玩家被困在一座废弃设施中，通过探索和解谜来拼凑故事并逃出设施。**不允许多存档**。这与"房间预制件切换"机制并不完全吻合，说明皇上的原型描述可能是对 Hollow Halls 机制的某种想象性重构，或参考了其他类似机制的游戏（如 Dead Cells 的房间生成方式）。策划案以"房间预制件切换"为核心机制进行设计。
