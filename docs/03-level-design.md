# 关卡设计

## 3.1 地图/关卡结构

- **整体结构：** 类银河恶魔城的横向展开地图，非线性的区域连接
- **章节设计：** 共3个章节，每章5~8个房间
- **门控设计：** 完成上一章节全部房间后解锁下一章节
- **能力解锁：** 无（保持无成长驱动，所有机制在第一章逐步引入）

## 3.2 章节规划

### 第一章·觉醒
**目标：** 引入基础切换（地板/墙），教学导向，5个房间

| 房间 | 教学目标 | 槽位配置 |
|------|--------|--------|
| 1-1 | 理解槽位概念 | 1个ToggleSlot，2选项（墙/地板），切换后即可通行 |
| 1-2 | 理解有多个槽位 | 2个ToggleSlot，需同时切换正确组合 |
| 1-3 | 理解出口位置需要特定配置 | 2个ToggleSlot + 1个CycleSlot，出口被实墙堵住 |
| 1-4 | 首次失败教学：按R重置，回顾观察 | - |
| 1-5 | 引入CycleSlot | - |

### 第二章·深掘
**目标：** 引入机关门、时序开关（某槽位切换后另一槽位状态改变），6个房间
- ConditionalSlot（条件解锁）
- 时序依赖
- 门控

### 第三章·迷途
**目标：** 引入联动开关（需要同时满足多个槽位配置）、伪装地板（视觉欺骗），8个房间
- 联动开关
- 伪装地板（Crumbling/Fake）
- 空间欺骗
- 综合挑战

## 3.3 难度曲线设计

### 槽位复杂度升级路径
- **槽位数量：** 1 → 2 → 3 → 4（逐章增加）
- **选项数量：** 2 → 3 → 4（选项越多推理越复杂）
- **联动关系：** 无关 → 简单依赖 → 双向联动

### 前3个房间的教学引导

| 房间 | 教学目标 | 槽位配置 |
|------|--------|--------|
| 1-1 | 理解槽位概念 | 1个ToggleSlot，2选项（墙/地板），切换后即可通行 |
| 1-2 | 理解有多个槽位 | 2个ToggleSlot，需同时切换正确组合 |
| 1-3 | 理解出口位置需要特定配置 | 2个ToggleSlot + 1个CycleSlot，出口被实墙堵住 |

## 3.4 房间复杂度升级路径

| 参数 | 变化 |
|------|------|
