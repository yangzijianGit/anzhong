# 开发计划

## 开发周期总览

| 阶段 | 内容 |预计时间 |
|------|------|--------|
| 第1个月 | 核心机制 + 8个房间 | 4周（个人solo） |
| 第2个月 | 美术资源 + 音效 + 关卡编辑器 | 4周 |
| 第3个月 | 完整19个房间 + 第三章 + 发布准备 | 4周 |

## 第一阶段（第1个月）：核心机制 + 10个房间

**目标：** 可玩的Demo级别

- [ ] 搭建Unity 2D项目（URP + 2D Renderer）
- [ ] 实现Tilemap地板/墙壁渲染
- [ ] 实现SwitchSlot系统（Toggle / Cycle）
- [ ] 实现玩家移动（2D Movement，无战斗）
- [ ] 实现连通性验证（视觉反馈）
- [ ] 设计并实现第1章5个房间
- [ ] 设计并实现第2章前5个房间
- [ ] 实现R键重置功能
- [ ] 章节解锁系统
- [ ] 基础UI（房间标题）

## 第二阶段（第2个月）：美术资源 + 音效 + 编辑器

**目标：** 完整的视觉体验

- [ ] 制作正式美术资源（地板、墙壁、槽位光效、出口光效）
- [ ] 制作槽位UI（切换提示、当前选项图标）
- [ ] 收集/制作音效（切换、地板铺设、出口连通）
- [ ] 制作或生成BGM（ambient风格）
- [ ] 开发自定义关卡编辑器（Unity EditorWindow）
- [ ] 使用编辑器重做所有房间数据
- [ ] 添加存档系统（简单JSON本地存储）

## 第三阶段（第3个月）：完整章节 + 发布准备

**目标：** Steam/Itch.io发布

- [ ] 设计并实现第2章剩余房间 + 第3章全部房间（14个房间）
- [ ] 添加叙事碎片（环境文件/涂鸦）
- [ ] 添加成就系统（可选）
- [ ] 添加主菜单 + 设置界面
- [ ] 打包测试（Windows/Mac）
- [ ] 编写商店页文案、录制宣传视频
- [ ] 发布Itch.io（免费）或Steam（$4.99）

## MVP定义（第一个可玩版本）

### MVP内容（4周开发周期，个人solo）

| 内容 | 说明 |
|------|------|
| **核心机制** | SwitchSlot切换、地板/墙切换、连通性验证 |
| **房间数量** | 8个房间（第1章5个 + 第2章前3个） |
| **预制件类型** | SolidWall / Floor / Door |
| **章节系统** | 章节切换，房间解锁 |
| **重置功能** | R键重置当前房间 |
| **UI** | 简单房间标题显示、重置按钮 |

### 不包含

- 剧情叙事元素
- 成就/收集系统
- 第二章后半+第三章全部房间
- 完整音效和BGM
- 自定义关卡编辑器
- 存档系统
