# 流程设计

## 4.1 玩家操作流程图

```
┌─────────────────────────────────────────────────────────────┐
│                        主菜单                                 │
│                    [开始游戏] [设置]                        │
└───────────────────────┬───────────────────────────────────────┘
                        │
                        ▼
┌─────────────────────────────────────────────────────────────┐
│                     章节选择                                  │
│              [第一章·觉醒] [已解锁的第二章]                   │
└───────────────────────┬───────────────────────────────────────┘
                        │
                        ▼
┌─────────────────────────────────────────────────────────────┐
│                      进入房间                                │
│                   显示房间名称+目标                           │
└───────────────────────┬───────────────────────────────────────┘
                        │
                        ▼
┌─────────────────────────────────────────────────────────────┐
│                       游戏中                                  │

│  │ 移动/WASD│ ──► 槽位位置/ ──► │ E键切换  │ ──► 改变房间拓补  │

│        │                         │                              │
│        ▼                         ▼                              │
│  ┌──────────┐            ┌──────────┐                        │
│  │ 走到出口 │ ◄─────────│ 尝试行走 │                        │
│  │ R键重置  │            └──────────┘                        │
│  └──────────┘                                                   │
└───────────────────────┬───────────────────────────────────────┘
                        │
             ┌────────────┴────────────┐
             ▼                         ▼
┌─────────────────────┐     ┌─────────────────────┐
│     通关成功         │     │      卡住了          │
│  进入下一房间/章节  │     │    按R重置          │
└─────────────────────┘     └─────────────────────┘
```

```
┌─────────┐   开始游戏    ┌─────────┐   通关房间    ┌──────────┐
│ MAIN    │ ──────────► │ CHAPTER │ ─────────► │ ROOM     │
│ MENU    │              │ SELECT  │             │ PLAYING  │
└─────────┘              └─────────┘             └────┬─────┘
     ▲                                             │
     │                                             │ 暂停
     │                      ┌──────────────────────┴──┐ │
     │                      ▼                          ▼
     │                 ┌─────────┐              ┌─────────┐
     │                 │ WIN     │              │ PAUSE   │
     │                 │ (通关)  │              │ MENU    │
     │                 └────┬────┘              └────┬─────┘
     │                      │                        │
     │             章节完成/                     按ESC/

     │
     │                      │

（注：无LOSE状态，因为无失败）
```

## 4.3 房间内部流程

```
[进入房间] → [显示房间名称+教学提示(如需)] → [玩家自由探索]
                                              │
                                              ▼
                               ┌─────────────────────────────────┐
                               │      玩家循环操作                │
                               │  ① 观察（槽位位置/可选配置）    │
                               │  ② 走近槽位                    │
                               │  ③ 按E循环切换 / 按Q反向切换    │
                               │  ④ 观察房间结构变化            │
                               │  ⑤ 尝试走到出口               │
                               └─────────────────────────────────┘
                                              │
                              ┌────────────────┴────────────────┐
                              ▼                                 ▼
                    ┌─────────────────┐              ┌─────────────────┐
                    │  路径可达        │              │   路径仍封闭     │
                    │  [出口发光+音效] │              │   [玩家继续尝试] │
                    │  走到出口=通关   │              │   或按R重置      │
                    └─────────────────┘              └─────────────────┘
```

## 4.4 按键映射

| 按键 | 功能 |
|------|------|
| WASD/方向键 | 移动 |
| E键 | 切换选中槽位（顺时针） |
| Q键 | 切换选中槽位（逆时针） |
| R键 | 重置当前房间所有槽位 |
| ESC键 | 暂停菜单 |
