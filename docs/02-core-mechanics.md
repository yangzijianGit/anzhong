# 核心玩法

## 2.1 主要机制

| 要素 | 说明 |
|------|------|
| **预制件槽位（SwitchSlot）** | 房间中特定位置设有槽位，每个槽位包含2~4 个预制件选项 |
| **预制件类型** | 地板（可通过）/实墙（不可通过）/机关门（满足条件开启）/透明墙（可穿行） |
| **玩家操作** | 走近槽位 → 屏幕提示当前选项 → 按键循环切换预制件 → 房间结构实时变化 |
| **通关条件** | 切换出正确配置，使玩家可以从入口到达出口 |
| **失败条件** | **无**——可无限切换，可随时重置（Reset）房间 |
| **重置机制** | 按R键将所有槽位恢复到房间初始状态 |

## 2.2 核心循环

```
[进入房间] → [观察环境/槽位位置] → [尝试切换1个或多个槽位] → [验证连通性（视觉判断）] → [到达出口] → [进入下一房间]
```

-核心循环强调**即时反馈**：每次切换后，玩家立刻能看出道路是否打通
- 验证连通性是纯视觉的（无需寻路算法提示），通过地板颜色/发光指引

## 2.3 槽位类型（SwitchSlot）

| 类型 | 行为 |
|------|------|
| **ToggleSlot** | 二选一切换（A↔ B） |
| **CycleSlot** | 循环切换（A → B → C → A） |
| **ConditionalSlot** | 依赖另一槽位状态，解锁更多选项 |
| **LockedSlot** | 需要先完成特定目标才可激活 |

## 2.4 预制件类型（PrefabOption）

| 类型 | 视觉效果 | 行为 |
|------|--------|------|
| **SolidWall** | 深灰色实墙 | 阻挡移动 |
| **Floor** | 地板色，可通行 | 正常地面 |
| **GlassWall** | 半透明蓝色 | 可通行但视觉上有阻挡感 |
| **Door** | 门框（开/闭状态） | 关闭时不可通行，开启后可通行 |
| **CrumblingFloor** | 碎裂地板纹理 | 踩上去后碎裂消失（一次性） |
| **FakeFloor** | 与普通地板外观相同 | 实际为实墙，玩家会踩空/视觉反馈 |
| **PressurePlate** | 踩下时触发 | 联动其他槽位变化 |

## 2.5 切换动画和反馈

- **切换动画：** 预制件淡出淡入（0.2s），伴随切换音效
- **视觉反馈：** 槽位位置有微弱发光标记（引导玩家注意力）
- **连通性反馈：** 当出口路径连通时，出口发出脉冲光提示

## 2.6 按键映射

| 按键 | 功能 |
|------|------|
| WASD/方向键 | 移动 |
| E键 | 切换选中槽位（顺时针） |
| Q键 | 切换选中槽位（逆时针） |
| R键 | 重置当前房间所有槽位 |
| ESC键 | 暂停菜单 |
