# 《暗室》游戏概述

> **一句话描述（≤30字）：** 无战斗 2D 房间解谜，切换槽位重塑空间开辟通路。
>
> **30 秒宣传语：** 在废弃的研究设施中，每一次切换都可能改变一切。

本文档是《暗室》产品文档体系的**总览与对齐基线**。它向新加入的工程师/美术/策划/老板/玩家**用 5 分钟讲清**：本游戏是什么、不是什么、为什么好玩、怎么实现、成本多少、风险在哪。其他 11 份文档（02 ~ 12）均以此文档为顶层约束。

## 一句话描述 (One-liner)

> **"无战斗 2D 房间解谜，切换槽位重塑空间开辟通路。"（22字）**

扩展版（用于 Steam 描述）：《暗室》是一款 2D 房间解谜游戏。玩家在废弃的研究设施中，通过切换房间内的"预制件槽位"（SwitchSlot）来重塑空间结构，开辟通往出口的道路。**无战斗、无成长、无死亡**——核心体验是**空间逻辑推理 + 房间拓扑变换**带来的顿悟快感。

## 目标用户 (Target Audience)

| 维度 | 具体化描述 |
|------|----------|
| **年龄** | 25 ~ 40 岁 |
| **游戏经验** | 玩过 Gorogoa / The Pedestrian / Viewfinder / Inside / Limbo 中至少 1 款 |
| **动机** | 追求"顿悟快感"与"独处 1-2 小时静心解谜"的休闲解谜玩家 |
| **行为习惯** | 喜欢 Steam 探索队列、能接受 $5 价位、偏好单机短篇体验 |
| **不喜欢的玩家** | 追求战斗刺激的、追求数值成长的、追求剧情叙事的、追求多人联机的 |

**为什么不是"广大玩家"：** 本游戏无战斗、无成长、无叙事，是"硬核休闲解谜"窄品类。面向广泛玩家会导致每类用户都觉得"差点意思"。聚焦于"解谜 + 空间逻辑"这个具体人群，转化率与口碑更高。

## 核心特色 (Core Features)

| # | 特色 | 验收标准（可独立验证） |
|---|------|----------------------|
| 1 | **无战斗** | 全流程无 HP 数值、无敌人 NPC、无战斗碰撞回调、无战斗音效 |
| 2 | **无成长** | 无经验值、无等级、无技能树、无装备系统——所有机制在 Ch1 全部引入 |
| 3 | **无死亡惩罚** | 无失败状态；玩家可无限切换槽位、可随时 Reset 房间，无加载时间 |
| 4 | **即时反馈** | 每次切换后 ≤ 200ms 视觉反馈（槽位发光增强 + 预制件形变动画） |
| 5 | **空间逻辑推理** | 19 个房间中 ≥ 15 间需要"先观察拓扑结构 → 推理配置 → 切换验证"的认知循环 |

## 差异化卖点 (USPs)

> **对比至少 2 款同类游戏，明确"我们独特在哪"。**

| 维度 | 《暗室》 | The Pedestrian | Viewfinder | Gorogoa |
|------|---------|----------------|------------|---------|
| **视角** | 2D 侧视 | 3D 拼贴 | 1 人第 3 人称 | 2D 平面拼贴 |
| **核心机制** | 切换槽位重塑空间 | 拼接管线 | 拍照重置视角 | 拖动拼贴图层 |
| **战斗/成长** | 无 | 无 | 无 | 无 |
| **重塑空间** | ✅ 房间内结构变化 | ❌ 静态场景 | ❌ 视角变化 | ❌ 图层移动 |
| **房间数** | 19 间 | 35+ 关 | 70+ 关 | 30+ 屏 |
| **通关时长** | 3-5 小时 | 6-8 小时 | 8-10 小时 | 4-5 小时 |
| **定价** | **$4.99** | $9.99 | $19.99 | $9.99 |
| **独特定位** | **2D + 空间结构重塑 + 低价** | 3D 管线美学 | 照片视角革命 | 手绘拼贴艺术 |

**我们独特在哪：**

1. **2D + 空间重塑** — 同时具备 2D 的简洁与"房间结构变化"的玩法纵深，The Pedestrian / Viewfinder 都做不到
2. **低价 + 高完成度** — $4.99 提供 3-5 小时完整流程，比 The Pedestrian 便宜 50%
3. **无失败压力** — 相比 Gorogoa 偶尔的"卡关无解"挫败感，本游戏 Reset 房间 0 成本

## 乐趣点 (Why Fun?)

| # | 乐趣点 | 触发场景 | 期望情绪 |
|---|--------|---------|---------|
| 1 | **顿悟快感** | 玩家在 Ch1-5 第一次"切换后路径打通"瞬间 | "啊哈！"、嘴角上扬 |
| 2 | **空间逻辑推理** | 玩家盯着房间拓扑图 30 秒后找到唯一解 | 专注、心流 |
| 3 | **零压力** | 玩家在 Ch2-3 反复 Reset 不带挫败感 | 放松、自由探索 |
| 4 | **30 分钟节奏** | 每章 5-8 间房、单关 5-15 分钟 | 短时段可完成 |
| 5 | **视觉欺骗惊喜** | 玩家在 Ch3 遇到"视觉对称但实际不对"的隐藏关 | 惊喜、会心一笑 |

## 上手难度 (Onboarding)

| 维度 | 目标 | 验证方式 |
|------|------|---------|
| **第 1 间教学房 (1-1)** | ≤ 2 分钟内完成第一次切换并走到出口 | Playtest 计时 |
| **30 分钟内** | 掌握全部 4 种槽位类型（Toggle / Cycle / Conditional / Locked） | Playtest 答题 |
| **第 1 章通关** | ≤ 30 分钟（5 房间） | Playtest 计时 |
| **教学节奏** | "先做后讲"：1-1 不解释，玩家试错一次后 1-2 出提示 | 关卡文档追踪 |

## 弃坑风险 (Anti-Churn Points)

> 玩家会在哪些设计点上 10 分钟内弃坑？设计时主动规避。

| 风险点 | 触发条件 | 缓解设计 |
|-------|---------|---------|
| 教程太长 | 第 1 间 > 5 分钟未通关 | 1-1 无文字教学，纯试错 |
| 第 1 间无反馈 | 按 E 后无视觉变化 | 切换必有 200ms 发光增强 + 形变动画 |
| 无 Reset 提示 | 玩家不知可重置 | 1-1 第 1 次切换失败后 3 秒弹出 Reset 提示 |
| 价格高 | > $9.99 | 锁定 $4.99，参考 The Pedestrian 50% off |
| 无重玩价值 | 通关即弃 | 通关后解锁"最少步数挑战" + 隐藏成就 |

## 整体情绪曲线 (Emotional Arc)

```
好奇探索 ──→ 挑战顿悟 ──→ 空间颠覆 ──→ 回味成就感
  (Ch1)        (Ch2)         (Ch3)         (通关)
  +3~+5        +2~+6         -2~+8         +7~+8
```

- **Ch1 觉醒（前 30 分钟）：** 好奇 → 顿悟感（明亮蓝灰主调，#1A1A2E 背景）
- **Ch2 深掘（30-90 分钟）：** 挑战 → 思考压迫（压抑 + 局部动态光，雾效渐增）
- **Ch3 迷途（90-180 分钟）：** 迷失 → 空间颠覆（强光影对比 + 视觉欺骗）
- **通关：** 成就感 + 回味（章节回顾动画 + 步数展示）

## 技术栈 (Tech Stack)

> 顶层技术约束，与 02-core-mechanics.md 和 12-art-style.md 一致。

| 维度 | 选型 | 理由 |
|------|------|------|
| **引擎** | Unity 2022 LTS | 稳定 LTS、2D 支持完善、Asset Store 资源丰富 |
| **渲染** | URP + 2D Renderer | 支持 2D 光照、性能优于 Built-in |
| **地图** | Unity Tilemap | 基础地板/墙用 Tilemap，槽位用 GameObject 叠加 |
| **物理** | Unity 2D Physics（无 Rigidbody） | 静态碰撞用 Tilemap Collider，动态碰撞用 2D Collider |
| **动画** | Unity Animator + DOTween | 切换动画与 UI 反馈 |
| **输入** | 键盘 + Xbox/PS 手柄 | Input System（新） |
| **存档** | JSON（Resources.Load） | 人类可读、易调试 |
| **平台** | Windows / macOS / Linux | Steam 首发；WebGL 不在 v1.0 |
| **第三方依赖** | DOTween（MIT） / 其余 0 依赖 | 自带 Audio / 不用 FMOD |

|------|------|---------|

| **单场景 DrawCall** | ≤ 50 | Frame Debugger |

| **冷启动时间** | ≤ 5 秒到主菜单 | 计时 |

## 灵感来源与差异

| 参考游戏 | 我们借鉴了什么 | 我们不做什么 | 关键差异 |
|---------|--------------|------------|---------|
| **Gorogoa** | 平面几何 + 空间切换 + 无文字叙事 | 不做手绘拼贴，做 2D 像素 | 我们像素 + 房间重塑 vs 它手绘 + 图层平移 |
| **Dead Cells** | 房间预制件 + 像素渲染 + 干净轮廓 | 不做战斗 + Rogue-like | 我们固定 19 房间 vs 它随机生成 |
| **The Pedestrian** | 极简 UI + 无文字 + 信号美学 | 不做 3D 拼贴，做 2D 房间 | 我们 2D vs 它 3D，价格 50% off |
| **Inside** | 黑暗氛围 + 剪影 + 单光源 | 不做叙事电影化，做解谜 | 我们解谜驱动 vs 它叙事驱动 |
| **Hollow Halls** | 调研后发现：恐怖氛围 + 故事 + 探索 | 不做恐怖、不做叙事 | 我们神秘探索 vs 它恐怖惊吓 |

**结论：** 我们做的是 **"2D + 房间预制件切换 + 极简解谜 + 低价"** 的独特定位，避开与 3D 大作的正面竞争。

## 边界条件 (Edge Cases)

> 列 5+ 条 edge case，含触发条件与预期行为。

1. **玩家在切换动画中退出房间（按 ESC）**
   - 触发条件：动画未完成时按 ESC 触发退出
   - 预期行为：动画立即完成（不卡死），回到主菜单，槽位状态不存档

2. **帧率掉到 30FPS（性能边界）**
   - 触发条件：低端机或后台进程占用 CPU
   - 预期行为：切换动画时长自动拉长到 2x（200ms → 400ms），保持视觉完整

3. **ConditionalSlot 依赖对象不存在**
   - 触发条件：玩家先 Reset 房间使某个对象消失，ConditionalSlot 仍处于"激活"态
   - 预期行为：ConditionalSlot 自动回退到 inactive 态，预制件恢复初始

4. **玩家快速连按切换键（每秒 10 次）**
   - 触发条件：玩家测试或焦虑时连按
   - 预期行为：300ms 内只触发 1 次切换（防误触），其余按键忽略

5. **存档损坏（JSON 解析失败）**
   - 触发条件：磁盘错误或玩家手动修改存档
   - 预期行为：自动备份最近一次有效存档，降级为"无存档模式"，提示玩家

6. **玩家在 Ch3 视觉欺骗房间被误导（配置"看起来对"实际"错"）**
   - 触发条件：玩家基于视觉对称推理但实际不对称
   - 预期行为：3 次错误配置后，槽位发出"暗淡脉冲"（-50% 亮度）作为"方向不对"提示
