# UI/UX设计

## 8.1 核心HUD布局

```
┌─────────────────────────────────────────────────────────┐
│ [房间ID]                    [重置:R]            [ESC:设置] │
├─────────────────────────────────────────────────────────┤
│                                                         │
│                                                         │
│                         房                               │
│                         间                               │
│                         区                               │
│                                                         │
│                          ←玩家(青色小光点)               │
│                                                         │
│                                          [出口]→        │
│                                     (连通时发光)       │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**HUD位置：**
- 左上角：房间名称 + 章节进度（如 Ch1-3/5）
- 右上角：设置按钮 + 帮助按钮
- 右下角：小键盘提示（E切换/Q反向/R重置）

## 8.2 槽位交互UI

**槽位提示（玩家靠近时显示）：**

```
┌─────────────────────────────────┐
│  ◉ 槽位 1/3          [E]切换   │
│  当前: 地板        [Q]反向    │
│  ━━━━━━━━━━                   │
│  ① 地板 ② 墙 ③ 门          │
│  ● ○ ○                         │
└─────────────────────────────────┘
```

**设计原则：**
- 玩家距离槽位2格内时显示提示
- 半透明黑色底+白色文字
- 当前选中项高亮显示
- 切换后提示淡入淡出

## 8.3 界面视觉规范

| 元素 | 颜色/规格 |
|------|----------|
| 主文字 | #E0E0E0 (浅灰) |
| 强调文字 | #00D4FF (青色) |
| 警告/提示 | #FF9500 (橙色) |
| 背景面板 | rgba(0,0,0,0.6) 半透明黑 |
| 字体 | 无衬线字体（如Inter、Noto Sans） |
| 文字大小 | 房间名18px / HUD 14px / 提示 12px |

## 8.4 交互反馈

| 场景 | 视觉反馈 | 声音反馈 |
|------|----------|----------|
| 槽位高亮 | 青色脉冲发光 | 微弱的嗡鸣声 |
| 切换成功 | 淡入淡出动画 | 电子音"咔哒" |
| 路径连通 | 出口橙色脉冲 | 上升音提示 |
| 重置 | 闪白一下 | 短促的卷轴倒带声 |
