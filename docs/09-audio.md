# 音效与音乐需求清单

## 9.1 音效清单

| 事件 | 描述 | 时长 | 参考风格 |
|------|------|------|----------|
| **切换成功** | 机械卡扣声 | 0.2s | 电子音+机械感 |
| **多个槽位切换** | 连续咔哒声(0.1s间隔) | 0.4s | 机关启动 |
| **重置** | 卷轴倒带/数据清除声 | 0.3s | 科技感 |
| **出口连通** | 通电/上升音效 | 0.5s | 成就感提示音 |
| **踏入无效区域** | 轻微阻碍音/撞击音 | 0.15s | 轻柔的拒绝音 |
| **进入新房间** | 开门/空间转换 | 0.6s | 场景转换 |
| **章节完成** | 完成音效 | 1.0s | 成就结算 |
| **获得成就** | 解锁音 | 0.8s | 正向激励 |

## 9.2 环境音效

| 场景 | 描述 | 循环方式 |
|------|------|----------|
| **室内背景** | 低沉的设施环境音（嗡嗡声、远处水滴） | 循环 |
| **走廊回声** | 脚步回声（如果有） | 触发 |
| **电力蜂鸣** | 槽位附近的电流声 | 循环（靠近时渐强） |

## 9.3 音乐需求

| 场景 | 风格 | 参考 | 长度 |
|------|------|------|------|
| **主菜单** | Ambient+神秘感 | Gorogoa OST | 循环60s |
| **第一章** | 平静探索 | 科幻设施ambient | 循环90s |
| **第二章** | 紧张+好奇 | 更低沉、短促 | 循环90s |
| **第三章** | 悬疑+深暗 | 最低沉、有节奏 | 循环90s |
| **通关** | 释然+成就感 | 温暖的尾声 | 30s |

**BGM技术方案：**
- 使用Suno/Udio生成（搜索标签：ambient, mysterious, calm, facility）
- 或使用免费素材：OpenGameArt Audio专区

## 9.4 音频工具链

| 用途 | 工具 |
|------|------|
| 音效收集 | Kenney.nl (免费CC0)、freesound.org |
| 音乐生成 | Suno/Udio (付费) 或 付费素材包 |
| 音频编辑 | Audacity (免费) |
| 音效合并 | Audiosprite (Web) |
| 音量规范化 | LUFS Meter |

## 9.5 最低成本方案

- **免费音效库：** Kenney.nl（免费游戏素材）、freesound.org（CC0音效）
- **AI生成音乐：** Suno / Udio（生成氛围感BGM，风格tag：ambient, mysterious, calm）
- **工具链：** Audacity（音频剪辑）+ Audiosprite（音效合并）
