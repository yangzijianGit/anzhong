# 《暗室》策划文档

> **一款2D类银河恶魔城无战斗房间解谜游戏。玩家通过切换房间内的"预制件槽位"（SwitchSlot）来重塑房间结构，开辟通往出口的道路。无敌人、无战斗、无升级成长，核心体验是"空间逻辑推理 + 房间拓扑变换"。**
