# 《暗室》乐器与音色（design/music/instruments.md）

> **一句话定位：** 9 类音频 → 乐器对应矩阵 + 15 主乐器清单 + 5 段落主乐器 + 19 房间主乐器 + 音色库来源 + 6 阶段制作流程。

## 目的 (Purpose)

本文档是《暗室》音乐层**乐器选型与音色库**的完整定义。它向：

- **音乐总监 / 音频工程师** — 讲清**9 类音频对应乐器** + **15 主乐器清单** + **5 段落主乐器** + **19 房间主乐器** + **音色库来源**
- **外包商** — Kenney.nl CC0 + 自制 7 预制件 + 第三方授权 6 来源的可用素材清单
- **Unity 工程师** — `AudioClip` 命名 / `AudioMixer` 分组 / 音色文件导入路径
- **QA / 测试** — 6 阶段制作流程（草图 → 采样 → 编辑 → 混音 → 集成 → 测试）的验收基线

**本文档范围：** **乐器（Instrument）+ 音色（Timbre）**——具体的乐器选型、音色库来源、制作流程；不涉及风格 / 混音 / 编码（详见 `style.md` + `layer-mixing.md` + `standards.md`）。

## 1. 9 类音频对应乐器 (9 Audio Categories × Instruments)

> **核心设计原则：** 9 类音频（A1-A9）每类有**主乐器 + 辅乐器 2 种**——主乐器承担主要音色，辅乐器提供层次。

### 1.1 9 类音频 → 乐器对应矩阵 (9 Categories × Instruments)

| # | 资产类别 | 主乐器 | 辅乐器 | 时长 | 音量 dB | 文件来源 | 引用 |
|---|---------|--------|--------|------|--------|---------|------|
| **A1** | 切换音 (Switch SFX) | TS=大提琴拨奏 / CS=短笛 / CDS=钢琴高音 / LS=低音提琴 | 钹（条件满足时）| 0.20-0.40s | -12 / -18 | Kenney.nl / 自制 | 09-v2 §1.2 |

| **A3** | 通关音 (Win SFX) | 大提琴 + 钢琴 + 弦乐合奏 | 钟声（章节完成）+ 风（通关）| 0.50-8s | -6 / -3 | Kenney.nl / 自制 | 09-v2 §1.4 |

> **关键设计：** 9 类音频**乐器不重叠**——切换音用大提琴拨奏，重置音用低音提琴，通关音用弦乐合奏，避免"切换音与重置音混淆"。

## 2. 主乐器清单 (15 Main Instruments)

> **设计原则：** 15 种主乐器覆盖 9 类音频的所有音色需求——包括古典乐器（9 种）+ 电声乐器（3 种）+ 打击乐器（3 种）。

|---|------|------|---------|---------|------|

> **关键设计：** 人声采样**仅用于 TTS（无障碍）**，**绝不用于 BGM**——符合 SC-02 风格限制。

### 2.2 乐器族分类 (Instrument Family Classification)

```
弦乐家族 (String Family):
  - 大提琴 (01) - 中低频
  - 小提琴 (03) - 中高频
  - 竖琴 (05) - 拨奏
  - 低音提琴 (06) - 低频
  = 4 / 15 (27%)

键盘家族 (Keyboard Family):
  - 钢琴 (02) - 全频
  - 管风琴 (07) - 全频
  - 电钢琴 (08) - 全频
  - 钢片琴 (10) - 中高频
  = 4 / 15 (27%)

木管家族 (Woodwind Family):
  - 长笛 (04) - 中高频
  = 1 / 15 (7%)

打击家族 (Percussion Family):
  - 风铃 (09) - 极高频
  - 钹 (11) - 极高频
  - 锣 (12) - 中低频 + 长混响
  - 沙锤 (13) - 高频
  - 手鼓 (14) - 中低频
  = 5 / 15 (33%)

人声 (Vocal Family):
  - 人声采样 (15) - 仅 TTS
  = 1 / 15 (7%)
```

> **关键设计：** 打击乐占 33%（5/15）——dark ambient + puzzle 需要"保留打击感强化节奏"，但**复杂节奏除外**（SC-03 风格限制）。

## 3. 5 段落主乐器 (5-Section Main Instruments)

> **设计原则：** 5 段落（序章 / 第一章 / 间章 / 第二章 / 第三章）每段**1 主乐器 + 2 辅乐器 + 1 节奏乐器（可选）**。

### 3.1 5 段落主乐器对照表 (5-Section Main Instruments)

| 段落 | 章节 | 主乐器 | 2 辅乐器 | 1 节奏乐器（可选）| 音色特征 | 风格层 | 引用 |
|------|------|--------|---------|----------------|---------|--------|------|
| **序章** | 主菜单 | **大提琴** (01) | pad (合成器) + 风声 | 无 | 长混响 + 极远声场 | L1 dark ambient | 09-v2 §1.7 |
| **第一章** | Ch1 觉醒 | **钢琴** (02) | 小提琴 + 长笛 | 无（安静）| 极简 + 大留白 + 平静 | L3 minimalist | 09-v2 §1.7 |
| **间章** | 章节过渡 | **风铃** (09) | pad (合成器) + 钢琴高音 | 无 | 短动机 + 突变 + 留白 | L2 puzzle + L3 | 本文 §4.3 |
| **第二章** | Ch2 深掘 | **合成器** (不计入 15) | 低频 + 远处机械声 | 手鼓 (14) 50% 密度 | 低沉 + 不和谐 | L1 dark + L2 puzzle | 09-v2 §1.7 |
| **第三章** | Ch3 迷途 | **合成器 + 全乐器** | 鼓点 + 突变音 + 锣 (12) | 手鼓 (14) 100% + 沙锤 (13) | 不和谐 + 鼓点 + 突变 | L4 主题循环 + L2 puzzle | 09-v2 §1.7 |

> **关键设计：** 5 段落**主乐器强度递增**（序章 1 乐器 → 第三章 6+ 乐器），强化"难度递增 = 乐器复杂度递增"。

### 3.2 段落过渡的乐器渐变契约 (Section Transition Instrument Gradient)

| **Ch1 → 间章** | 钢琴 → 风铃（p fade out + p fade in）| 小提琴持续 + pad 渐入 | 3s cross-fade | 本文 §4.3 |

## 6. 音色制作流程 (Instrument Production Pipeline)

> **设计原则：** 6 阶段流程——**草图 → 采样 → 编辑 → 混音 → 集成 → 测试**——确保每个音色**符合规格基线**（09-v2 §1 dB + 音色）+ **通过验收**（AC-01~05）。

### 6.1 6 阶段制作流程表 (6-Stage Production Pipeline)

| 阶段 | 名称 | 输入 | 输出 | 工具 | 工时占比 | 验收 |
|------|------|------|------|------|:-------:|------|
| **01** | **草图 (Sketching)** | 设计意图 | 音色描述 + 参考作品 | 文档 + 参考 | 5% | 音色描述清单 |
| **02** | **采样 (Sampling)** | 音色描述 | 原始音频文件 | 自录音 + Spitfire LABS + Kenney.nl | 25% | 原始音频 100% |
| **03** | **编辑 (Editing)** | 原始音频 | 编辑后音频 | Audacity | 25% | 时长 / dB / 频谱符合 |
| **04** | **混音 (Mixing)** | 编辑后音频 | LUFS 规范化音频 | LUFS Meter | 15% | -16 LUFS 规范 |
| **05** | **集成 (Integration)** | LUFS 规范化音频 | AudioClip + AudioMixer 分组 | Unity Audio Import | 15% | Addressables 分组 |
| **06** | **测试 (Testing)** | AudioClip | 测试通过 | Profiler Audio | 15% | 同时发声 ≤ 8 + BGM ≤ 2 |

> **总工时：** 7 预制件 × 2h = **14h 自制工时**（不含 BGM 生成）+ 5 BGM × 3h (Suno/Udio + 编辑) = **15h BGM 工时** = **29h 总工时**（与 design/art/asset-list.md 14h + 数字画集 14h 类似）。

### 6.2 草图阶段 (Sketching)

- **任务：** 列出每个音色的**描述清单**（音色 + 时长 + dB + 触发条件）
- **示例：** `sfx_switch_toggle_01.wav`：短促电子咔哒 + 机械卡扣声 / 0.20s / -12dB / Kenney.nl 或自制 / CC0 / 按 E 切换 2 选项
- **工具：** Markdown 表 + 参考作品（Dear Esther / Gorogoa 等）
- **验收：** 音色描述清单完整（每音色 6 字段）

### 6.3 采样阶段 (Sampling)

- **任务：** 从音色库（Kenney.nl / Spitfire LABS / VSCO2 CE / 自录音）**采集**原始音频
- **示例：** 自制 `sfx_switch_toggle_01.wav` = 大提琴拨奏 + 钹一次（Audacity 多轨合成）
- **工具：** Audacity（多轨）+ Spitfire LABS（采样回放）+ 自录音（手机录音 / 电容麦克风）
- **验收：** 原始音频 100% 覆盖音色描述

### 6.4 编辑阶段 (Editing)

- **任务：** 编辑原始音频——**时长裁剪** + **频谱调整** + **静音去除**
- **工具：** Audacity
- **验收：** 时长符合（精确到 0.01s）+ dB 符合 + 频谱分析 100-8000Hz

### 6.5 混音阶段 (Mixing)

- **任务：** LUFS 规范化 + 音量标准化 + 静音头尾去除
- **工具：** Audacity + LUFS Meter
- **验收：** -16 LUFS（游戏标准）+ 头尾 100ms 静音 + 峰值 < -3dBFS

### 6.6 集成阶段 (Integration)

- **任务：** 导入 Unity + **Addressables 分组** + AudioMixer 分组 + 命名规范
- **工具：** Unity Editor + Addressables 1.21+ + AudioMixer.mixer
- **验收：** AudioClip ID 与 09-v2 §1.1 资产 ID 完全一致 + 命名规范（详见 `standards.md` §4）

### 6.7 测试阶段 (Testing)

- **任务：** 同时发声 ≤ 8 + BGM 通道 ≤ 2 + 文件总大小 ≤ 50MB + CPU ≤ 5% + 内存 ≤ 30MB
- **工具：** Unity Profiler Audio + Resources 统计
- **验收：** 性能约束 6 项（09-v2 §8 性能约束 6 项全部通过）

## 7. 边界条件 (Edge Cases)

1. **自制音色与 Kenney.nl 同名冲突**
   - **触发条件：** 自制 `sfx_switch_toggle_01.wav` 与 Kenney.nl `sfx_switch_toggle_01.wav` 同名
   - **优先级：** 自制版本（priority = 100）> Kenney.nl 版本（priority = 50）
   - **验收：** Addressables 分组 `audio/sfx/switch/` 自制版本优先

2. **Suno/Udio 生成 BGM 时长超出 90s**
   - **触发条件：** Suno/Udio 生成 BGM 时长 120s（超出 90s 上限）
   - **音频预期：** Audacity 剪辑至 90s + 自然循环点 + fade-in/out

3. **Suno/Udio 生成 BGM 风格不符（戏剧化太重）**
   - **触发条件：** Suno/Udio AI 生成风格过于激烈 / 不符合 dark ambient
   - **音频预期：** 重新生成 + 提供更详细 prompt（4 Pillar 风格约束），最多 5 次重试

4. **第三方授权过期（Suno/Udio 订阅停止）**
   - **触发条件：** Suno/Udio 订阅结束，商用协议终止
   - **音频预期：** v1.0 已发布的 BGM 不受影响（商用协议在订阅期间有效），新版本需重新生成

5. **CC-BY 素材署名要求导致发布复杂**
   - **触发条件：** v1.0 使用 CC-BY 素材，需在游戏内显示署名
   - **音频预期：** v1.0 全用 CC0 + 商用免费，避开 CC-BY 署名问题

6. **乐器素材版权变更（来源网站下架）**
   - **触发条件：** Kenney.nl / VSCO2 CE 网站下架导致部分素材丢失
   - **音频预期：** v1.0 截图存档 + 备份自制 7 预制件（10h 工时）
